# NL thesaurus

2. Dutch: Basiswoordenlijst Amsterdamse Kleuters (BAK)
   This is the "official" gold standard for Dutch vocabulary for ages 4-6. While often found in PDFs, researchers have put it into structured formats.

The List: Search for the BAK-lijst. It contains 3,000 words (2,000 core + 1,000 extension).

https://raw.githubusercontent.com/bzdgn/dutch-vocabulary/refs/heads/master/contents/01_WOORDEN.txt

Raw Data Alternative: The repository bzdgn/dutch-vocabulary contains a file called dutch-meaning.csv which is very simple and covers basic adjectives (happy, big, small) and nouns.

https://github.com/bzdgn/dutch-vocabulary
